# PusherMapperGUI
PixelPusher mapping GUI

## How to install
    git clone https://github.com/DecadentOasis/PusherMapperGUI.git
    cd PusherMapperGUI
    sudo npm install -g express
    npm install .

## How to run
    node index.js
Then point your browser at [http://localhost:3000](http://localhost:3000/)
